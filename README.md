# 🛠️ Sistema de HelpDesk em Java (CRUD em Console)

Este projeto é um sistema simples de HelpDesk desenvolvido em Java, com foco em aprendizado de lógica, orientação a objetos e operações CRUD.

## 📌 Funcionalidades

O sistema permite:

- Criar chamados
- Consultar chamados
- Editar chamados
- Excluir chamados
- Finalizar chamados
- Menu interativo no console

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Classes e Objetos
- Encapsulamento
- Listas (ArrayList)
- Estrutura de repetição (do-while)
- Estrutura condicional (switch-case)

## ▶️ Como executar

1. Abra o projeto em uma IDE (IntelliJ, Eclipse ou VS Code)
2. Execute a classe `Main.java`
3. Utilize o menu no console
